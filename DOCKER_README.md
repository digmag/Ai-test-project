# Hospital Management Application

Приложение для управления больницей, реализованное на Python с использованием фреймворка Tornado и базы данных Redis.

## Запуск с использованием Docker

Для запуска приложения с использованием Docker выполните следующие шаги:

### Требования

- Docker
- Docker Compose

### Запуск приложения

1. Клонируйте репозиторий (если еще не сделали этого)
2. Перейдите в директорию с проектом
3. Выполните команду:

```bash
docker-compose up -d
```

Приложение будет доступно по адресу: http://localhost:8888

### Структура сервисов

- `app`: Веб-приложение на Python (Tornado), запущенное на порту 8888
- `redis`: База данных Redis, запущенная на порту 6379

### Остановка приложения

Для остановки приложения выполните:

```bash
docker-compose down
```

### Перезапуск приложения

Для перезапуска приложения выполните:

```bash
docker-compose restart
```

### Просмотр логов

Для просмотра логов приложения:

```bash
docker-compose logs app
```

Для просмотра логов Redis:

```bash
docker-compose logs redis
```

### Пересборка образов

Если вы внесли изменения в код приложения и хотите пересобрать образ:

```bash
docker-compose build
docker-compose up -d
```

## Функциональность приложения

Приложение предоставляет следующие возможности:

- Управление больницами (создание, просмотр)
- Управление врачами (создание, просмотр, привязка к больнице)
- Управление пациентами (создание, просмотр)
- Управление диагнозами (создание, привязка к пациенту)
- Управление связями врач-пациент

## Архитектура

- Python 3.9
- Tornado (веб-фреймворк)
- Redis (база данных)
- HTML шаблоны с Bootstrap CSS
- JavaScript библиотека WOW.js для анимаций

## Тесты

Для запуска unit-тестов выполните:

```bash
# Внутри контейнера
docker-compose exec app python -m unittest test_app.py -v

# Или локально
python -m unittest test_app.py -v
```
